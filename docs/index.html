A reading is a recorded approach within 150 m of a timing point. "Median/Mean" are seconds early (−) or late (+).

The point. Bus operators across the West of England publish their live vehicle data, and WECA's Bus Service Improvement Plan sets a 95% punctuality target for the area, but there is no accessible, route-level public scorecard showing whether that target is being met. This site builds one from the operators' own open data.

An independent, non-commercial measurement of West of England bus punctuality, built only from operators' own public open data so anyone can see whether the 95% target is being met. Full method, sources and limitations are in the methodology, and all the code and data are on GitHub.
